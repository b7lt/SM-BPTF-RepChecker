# SM-BPTF-RepChecker
A SourceMod Plugin for TF2 servers that queries backpack.tf to fetch user ban/reputation data and auto-handle marked scammers. 
Successor to the [SteamRep Checker](https://forums.alliedmods.net/showthread.php?t=162517) plugin now that SteamRep has officially sunset.

Cvars, Commands, and Installation instructions can be found on the [AlliedModders thread](https://forums.alliedmods.net/showthread.php?p=2844954).
